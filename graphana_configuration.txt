Graphana configuration

1. Go to 
https://grafana.com/docs/grafana/latest/setup-grafana/installation/rpm/

2. Copy

[grafana]
name=grafana
baseurl=https://rpm.grafana.com
repo_gpgcheck=1
enabled=1
gpgcheck=1
gpgkey=https://rpm.grafana.com/gpg.key
sslverify=1
sslcacert=/etc/pki/tls/certs/ca-bundle.crt

3. Create that file and past the configuration on  it

sudo vim /etc/yum.repos.d/graphana.repo

4. Graphana download

sudo yum install grafana

5. Install polkit

sudo yum install polkit


5. Check graphana status

sudo service grafana-server status

6. Start graphana

sudo service grafana-server start


7. Stop graphana

sudo service grafana-server stop
